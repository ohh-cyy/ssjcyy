# 锦绣千年

织锦智慧创意平台网站雏形：四大名锦纹样衍生设计、千年时间线长廊、场景智能适配，并预留小艺 Agent 对话入口。

## 本地启动

```bash
npm install
npm run dev
```

浏览器打开终端提示的本地地址即可。

## 当前雏形包含

1. **首屏品牌 Hero** — 「锦绣千年」
2. **AIGC 衍生设计** — 选纹样 / 选载体 / 预览 mock
3. **时间线数字长廊** — 汉 → 宋 → 明清 → 当代
4. **场景智能适配** — 模拟上传与方案推荐
5. **锦绣 Agent 浮层** — 关键词对话演示（后续可对接小艺开放平台）
